TENER EN CUENTA LO SIGUIENTE:

1. INSTALAR PYTHON EN EL PC, revisar que si quede correctamente en las variables de entorno del PC.
2. INSTALAR ESTAS LIBRERIAS, SEGÚN LA VERSION VARIA LA FORMA DE INSTALAR (puede ser la primera o la segunda validar)

- pip install numpy pandas matplotlib seaborn scikit-learn openpyxl
- py -m pip install numpy pandas matplotlib seaborn scikit-learn openpyx


3. Se debe instalar jinja2 para renderizado de vistas:
py -m pip install jinja2


4. para subir el server se debe poner en los comando: py server.py

abrir la url que muestre en el navegador ejemplo: http://localhost:8000/

para que abra siempre debe estar arriba el servidor es decir presionar: py server.py y abrir la url en el navegador es decir: http://localhost:8000/


5. Para la creación del modelo se debe instalar :py -m pip install fastapi uvicorn pyngrok joblib
